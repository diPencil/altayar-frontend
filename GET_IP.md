# 🔧 كيفية الحصول على عنوان IP للاتصال من الموبايل

## المشكلة
عندما تحاول فتح التطبيق على الموبايل من Expo، قد تواجه خطأ "Network request failed" لأن عنوان IP في الكود غير صحيح.

## الحل السريع

### Windows:
```powershell
ipconfig | findstr /i "IPv4"
```

### Mac/Linux:
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

أو:
```bash
hostname -I
```

## تحديث IP في الكود

1. افتح `frontend/src/services/api.ts`
2. ابحث عن السطر:
   ```typescript
   return 'http://192.168.1.10:8082/api';
   ```
3. غيّر `192.168.1.10` إلى IP جهازك الحالي

## ملاحظات مهمة

1. **الموبايل والكمبيوتر لازم يكونوا على نفس الشبكة (WiFi)**
2. **Backend لازم يكون شغال** على `http://0.0.0.0:8082` أو `http://192.168.1.10:8082`
3. **Firewall** قد يمنع الاتصال - تأكد إن Port 8082 مفتوح
4. **IP قد يتغير** كل مرة تتصل فيها بشبكة WiFi مختلفة

## التحقق من الاتصال

بعد تحديث IP، جرب من المتصفح على الموبايل:
```
http://YOUR_IP:8082/health
```

لو شفت `{"status": "healthy"}`, معناه الاتصال شغال! ✅

## حل بديل: استخدام Expo Tunnel

لو IP بيتغير كتير، ممكن تستخدم Expo Tunnel:

```bash
npx expo start --tunnel
```

لكن ده أبطأ شوية من الاتصال المباشر.
